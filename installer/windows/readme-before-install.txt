SYSTEM REQUIREMENTS
-------------------
- Windows 10 (1809+) or Windows 11
- 64-bit processor
- 8 GB RAM minimum (16 GB recommended)
- 30 GB free disk space (for application + AI models)
- Internet connection (for model downloads)

INSTALLATION SIZE
-----------------
- Application: ~500 MB
- AI Models (optional): ~20-25 GB
  * You can choose to download models during installation
  * Or download them later from within the application

WHAT WILL BE INSTALLED
----------------------
- Python 3.11 (embedded, isolated installation)
- LALO AI Platform backend and frontend
- Required Python dependencies
- Optional: AI model files

FIRST RUN
---------
After installation, the first run will:
1. Install Python dependencies (3-5 minutes)
2. Initialize the database
3. Optionally download AI models (15-30 minutes)

You can skip model downloads and add them later.
